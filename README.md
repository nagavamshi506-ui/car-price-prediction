# 🚗 Car Price Prediction App

This is a Machine Learning web application built using **Streamlit** that predicts the price of a car based on user inputs like brand, model, year, engine, mileage, and more.

---

## 📌 Features

* Predict car price instantly 💰
* Simple and interactive UI
* Built with Machine Learning model
* Easy to use for beginners

---

## 🧠 Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn

---

## 📂 Project Structure

```
car-price-app/
│
├── app.py              # Streamlit app
├── model.pkl           # Trained ML model
├── columns.pkl         # Saved feature columns
├── requirements.txt    # Required libraries
```

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/car-price-app.git
cd car-price-app
```

---

### 2️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

### 3️⃣ Run the app

```
streamlit run app.py
```

---

## 🚀 Usage

1. Enter car details:

   * Brand
   * Model
   * Year
   * Fuel Type
   * Transmission
   * Engine CC
   * Mileage
   * Seating Capacity
   * Safety Rating
   * Body Type

2. Click **Predict Price**

3. View the predicted car price 🎯
